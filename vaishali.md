
# How to Use VS Code

## Getting Started

1. **Download and Install**
    - Visit [code.visualstudio.com](https://code.visualstudio.com)
    - Choose your operating system and install

2. **Open a Folder**
    - Click `File > Open Folder`
    - Select your project directory

3. **Create Files**
    - Right-click in the Explorer sidebar
    - Select `New File` and name it

## Key Features

| Feature | Shortcut |
|---------|----------|
| Command Palette | `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (Mac) |
| Find | `Ctrl+F` |
| Replace | `Ctrl+H` |
| Terminal | `Ctrl+` ` |
| Debug | `F5` |

## Essential Extensions

- **Prettier** - Code formatter
- **ESLint** - JavaScript linter
- **GitLens** - Git integration
- **Copilot** - AI-powered code completion

## Tips

- Use the Sidebar to navigate files
- Customize settings via `File > Preferences > Settings`
- Install extensions from the Extensions Marketplace
- Use integrated terminal for command-line tasks
